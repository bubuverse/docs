## 📈 Guild Levels & Features

Guilds level up by accumulating XP. Higher levels unlock more features:

| Level | XP Required | Features Unlocked                                                           |
| ----- | ----------- | --------------------------------------------------------------------------- |
| LV1   | —           | Tags, avatar, nationality, guild link, leaderboard                          |
| LV2   | 16,000      | Starlight Shard reward pool, guild check-in system                          |
| LV3   | 33,000      | Guild Challenge: Planet Hunt, increased shard rewards, custom avatar frames |
| LV4   | 65,000      | Guild shop, shard & NFT exchange for STC, guild PK system                   |
